### 4. Interactive Features

- **Comment system** - add comments to specific text sections (chat F).
- **Suggestion mode** - propose changes without directly editing
- **Mention system** - @mention other users in comments (optional)
- **Notification system** - alert users about mentions and document changes (optional)

### 5. Document Features

- **Rich text formatting** (bold, italic, headers, lists, etc.)
- **WYSIWYG editing experience** using Tiptap or similar editor
- **Block-based editing** similar to Notion (headings, paragraphs, code blocks)
- **Collaborative editing** with real-time cursor and selection tracking
- **Link sharing** with permission controls
- **Search functionality** across all documents
